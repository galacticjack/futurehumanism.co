Get Started in 5 Minutes

Here's how to run your first open source agent with OpenClaw:

1

Install OpenClaw

Run npm install -g openclaw to install the CLI globally.

2

Add Your API Key

Set your OpenAI or Anthropic key: openclaw config set provider.apiKey YOUR_KEY

3

Connect a Channel (Optional)

Add Telegram: openclaw config set channels.telegram.token YOUR_BOT_TOKEN

4

Start Your Agent

Run openclaw gateway start and your agent is live!
